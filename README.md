### ⚡ Minimum — CPU Mode

| Component | Requirement |
|:---|:---|
| **OS** | Windows 10 / Ubuntu 20.04 / macOS 12+ |
| **Python** | 3.10 or higher |
| **RAM** | **64 GB** minimum (model runs in system RAM) |
| **Storage** | 60 GB free (model weights ≈ 52 GB) |
| **CPU** | x86-64, 8+ cores recommended |
| **Internet** | Required for first-run model download only |

### 🚀 Recommended — CUDA GPU Mode

| Component | Recommendation |
|:---|:---|
| **GPU** | NVIDIA RTX 3090 / 4090 / A100 |
| **VRAM** | 24 GB minimum — 48 GB+ for full speed |
| **RAM** | 32 GB system RAM |
| **CUDA Toolkit** | 11.8 or 12.x |
| **NVIDIA Driver** | 525 or newer |
| **Storage** | 60 GB NVMe SSD |

> 💡 **Performance note:** CPU mode produces ~1–5 tokens/sec. A modern GPU delivers 30–80+ tokens/sec.

## 🚀 Installation

</div>

```bash
# ── Step 1: Clone ──────────────────────────────────────────────
git clone https://github.com/im-aswajith/sakura-ai.git
cd sakura-ai

# ── Step 2 (GPU users only): Install PyTorch with CUDA ─────────
pip install torch --index-url https://download.pytorch.org/whl/cu121

# ── Step 3: Install dependencies ───────────────────────────────
pip install -r requirements.txt

# ── Step 4: Launch Sakura 🌸 ───────────────────────────────────
python sakura_ai.py
```

> **First launch auto-sequence:**
> 1. 🔍 Detects your hardware (CPU or CUDA)
> 2. ⬆️ Upgrades `huggingface_hub` + `transformers` from source
> 3. 📥 Downloads the model (~52 GB) from Hugging Face
> 4. 🔄 Restarts cleanly and greets you
